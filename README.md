# Single-Page Resume Website

A clean, semantic HTML resume website built without CSS, focusing on proper HTML structure and readability.

## Project Overview

This project demonstrates a single-page resume website using only HTML5 semantic tags. The layout matches the provided reference design and includes all essential resume sections.

## Features

- **Semantic HTML5**: Proper use of semantic tags (header, main, section, article, footer)
- **Clean Structure**: Well-organized sections with clear hierarchy
- **Tables**: Professional presentation of experience and education data
- **Lists**: Organized skills and project details using unordered lists

## Setup Instructions

### Option 1: Local Setup

1. **Clone or Download the Repository**
   ```bash
   git clone https://github.com/idreamfyrei/html-resume.git
   cd html-resume
   ```

2. **Open the HTML File**
   - Simply double-click `index.html` in your file explorer, or
   - Right-click `index.html` → Open with → Your preferred browser

### Option 2: Live Server

   **Using VS Code Live Server Extension**
   - Install "Live Server" extension in VS Code
   - Right-click on `index.html`
   - Select "Open with Live Server"
